> A front-end utils tool

## Usage

```
import utils from '@lgd-utils/utils';

utils.regexpTest(1, 2); // false
utils.regexpTest(/^\d+$/, '45'); // true
utils.undefinedTo(1, 2); // 1
utils.undefinedTo(undefined, 4); // 4
```
